
# Task Manager
Developed a full-stack task management system using **React.js** and **Bootstrap** for the frontend, and **Java, Spring, and Spring Boot** for the backend. Implemented RESTful APIs to enable task creation, updating, deletion, and status tracking. Designed a responsive UI and ensured efficient client-server communication for real-time task management. Built the application with a scalable and maintainable architecture, supporting future enhancements.



## Features

- Easy Task Handler
- Readable Code
- friendly UI



## Run



Go to the project directory

```bash
  cd fronttask
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
Default port:

```bash
  http://localhost:3000
```

## Server

**Important Notes**

Make sure Java is installed:

```bash
  java -version
```

Make sure Maven/Gradle is installed:

```bash
  mvn -version
  gradle -version
```

Default port:

```bash
  http://localhost:8080
```
Start the server

```bash
  cd management.task
  mvn spring-boot:run
```
## Tech Stack

**Client:** React,TailwindCSS ,BootStrap

**Server:** Java , Spring ,SpringBoot
